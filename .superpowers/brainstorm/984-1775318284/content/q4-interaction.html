Q4. 노드 클릭 시 동작 — 워크플로우 노드를 클릭하면?

각 단계(노드)를 클릭했을 때 어떤 동작을 할지 선택해주세요.

A

바로 해당 메뉴 페이지로 이동

예: "IQC" 노드 클릭 → /material/iqc 페이지로 즉시 이동
가장 단순하고 빠른 접근. 워크플로우 페이지를 업무 시작점(런치패드)으로 활용.

B

상세 팝업 → 메뉴 이동 (추천)

노드 클릭 시 간단한 팝오버에 현재 건수, 최근 처리 이력, "이동" 버튼을 표시.
맥락 확인 후 이동할지 결정할 수 있어서, 불필요한 페이지 전환을 줄임.

C

카드 확장 → 전체 플로우 상세

카드 클릭 시 전체 화면으로 확장되어 상세 플로우(분기 포함)와 단계별 현황을 표시.
각 노드에서 다시 해당 메뉴로 이동 가능. 2단계 구조.
