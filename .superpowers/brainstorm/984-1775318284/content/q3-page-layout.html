Q3. 페이지 레이아웃 — 워크플로우 목록을 어떻게 배치할까요?

7개 워크플로우를 하나의 페이지에 보여주는 방식입니다.

A. 아코디언 리스트

워크플로우 제목을 세로로 나열, 클릭 시 흐름도 펼침

장점

  • 세로 공간 효율적
  • 관심있는 것만 펼침

단점

  • 전체 한눈에 안 보임

B. 대시보드 그리드 (추천)

카드 형태로 모든 워크플로우를 한 화면에 배치, 각 카드 안에 하이브리드 플로우

장점

  • 전체 현황 한눈에 파악
  • 대시보드처럼 실용적
  • 카드 클릭 시 상세 플로우 모달/페이지 이동

단점

  • 카드당 공간 제한

상단 탭으로 워크플로우 전환, 한 번에 하나의 상세 플로우만 표시

  • 전체 현황 비교 불가
  • 탭 전환 필요
