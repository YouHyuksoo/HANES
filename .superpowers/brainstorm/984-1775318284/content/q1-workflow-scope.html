Q1. 워크플로우 범위 — 어떤 업무 흐름을 포함할까요?

현재 HANES MES 메뉴 구조에서 식별된 주요 업무 프로세스입니다. 포함할 워크플로우를 선택해주세요.

1

자재관리 (Material)

입하 → 입고라벨 → IQC검사(합격/불합격) → 입고(입고/취소) → 출고요청 → 출고

2

생산관리 (Production)

월간계획 → 시뮬레이션 → 작업지시 → 생산실적(수동/설비) → 공정검사 → 완제품입고

3

품질관리 (Quality)

IQC → 공정검사 → OQC → 불량등록 → 재작업 → 재검사 → CAPA

4

출하관리 (Shipping)

포장실적 → 포장 → 파렛트 → OQC → 출하확정 → 출하이력

5

설비관리 (Equipment)

PM계획 → PM작업지시 → PM실행 → 일상점검 → 정기점검 → 점검이력

6

외주관리 (Outsourcing)

외주업체 → 외주발주 → 외주입고

7

소모품관리 (Consumables)

소모품마스터 → 라벨 → 입고 → 출고 → 재고 → 수명 → 장착

Tip

복수 선택 가능합니다. 1차로 핵심 워크플로우만 구현하고, 나머지는 추후 확장할 수도 있습니다.
