HANES PDA에 추가할 메뉴 후보

SMMEX 패턴 + HANES 도메인(하네스 제조)에 맞게 제안합니다. 원하는 메뉴를 선택해주세요.

1

생산실적 입력

작업지시 바코드 스캔 → 양품/불량 수량 입력 → 실적 등록. SMMEX의 Loading/Unloading에 해당.

2

작업지시 조회

오늘의 작업지시 목록 확인. 라인/공정별 필터링, 진척률 표시.

3

LOT 추적

자재/제품 LOT 바코드 스캔 → 전체 이력 조회 (입고→투입→생산→출하). SMMEX의 Check Reel 유사.

4

품질검사 (IQC/OQC)

검사 대상 바코드 스캔 → 검사항목별 합격/불합격 입력. SMMEX의 Full Check에 해당.

5

자재 청구

라인에서 필요한 자재를 창고에 요청. SMMEX의 Material Request에 해당.

6

라벨 출력

바코드 스캔 → 라벨 미리보기 → 프린터 출력 (자재/제품/출하 라벨).

7

소모품 불출

소모품 바코드 스캔 → 수량 입력 → 불출 등록. 설비 교체 소모품 관리.

8

창고 이동

LOT 바코드 스캔 → 출발/도착 창고 선택 → 이동 처리. SMMEX의 Location Check 유사.

9

불량 등록

생산 중 불량 발생 시 바코드 스캔 → 불량유형 선택 → 수량 입력.

10

재작업 스캔

재작업 지시 바코드 스캔 → 공정별 실적 입력 → 완료 처리.
