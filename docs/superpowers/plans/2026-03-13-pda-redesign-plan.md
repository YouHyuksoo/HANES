# PDA 모바일 앱 구현 계획

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** HANES MES PDA 모바일 앱 7개 프로그램 + 프레임워크 구현 (기존 PWA 확장)

**Architecture:** 기존 Next.js PWA `/pda/` 라우트 확장. 백엔드 NestJS에 PDA 전용 API 추가. SMMEX PDA 프로젝트의 UI 컨셉(lucide-react 아이콘, 세로 리스트/2열 그리드, PageHeader 패턴) 참조. Zebra TC21 DataWedge는 키보드 웨지(HID) 모드로 기존 ScanInput이 처리.

**Tech Stack:** Next.js 15.2 / React 19 / Tailwind 4 / Zustand / React Query / NestJS / TypeORM / Oracle DB

**Spec:** `docs/superpowers/specs/2026-03-13-pda-redesign-design.md`

```typescript
// apps/backend/src/common/guards/inventory-freeze.guard.ts
@Injectable()
export class InventoryFreezeGuard implements CanActivate {
  constructor(
    @InjectRepository(MatPhysicalInv)
    private readonly physicalInvRepo: Repository<MatPhysicalInv>,
  ) {}

  async canActivate(context: ExecutionContext): Promise<boolean> {
    const activeInv = await this.physicalInvRepo.findOne({
      where: { status: 'IN_PROGRESS' },
    });
    if (activeInv) {
      throw new BadRequestException('재고실사 진행 중입니다. 자재 트랜잭션이 제한됩니다.');
    }
    return true;
  }
}
```
